## E-axis rollout contract

`wrangler.toml` now declares:

```toml
E_AXIS_STARTED_AT = "2026-07-13T12:22:37.508657Z"
E_AXIS_SHADOW_DAYS = "7"
E_AXIS_RANKING_ENABLED = "false"
```

The shadow window becomes review-ready after seven elapsed days, but ranking applies only when both conditions are true:

1. the shadow window has elapsed;
2. `E_AXIS_RANKING_ENABLED` is explicitly set to `true` in a reviewed deployment.

This avoids both indefinite shadow and surprise automatic activation.
